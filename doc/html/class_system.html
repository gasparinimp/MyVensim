Detailed Description

Representa um compartimento (stock) no modelo de simulação dinâmica.

Armazena um valor numérico que é modificado a cada passo de simulação pelos fluxos (Flow) conectados a ele. Equivale ao conceito de "Stock" nos diagramas de Vensim/Forrester.
