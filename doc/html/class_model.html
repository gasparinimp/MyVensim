Detailed Description

Coordena Systems e Flows e executa a simulação dinâmica.

Agrega coleções de Systems (estoques) e Flows (fluxos) e implementa o método run(), que aplica a atualização assíncrona (Euler explícito) a cada passo de tempo.

◆ run()

void Model::run ( int t_initial,
int t_end )

Executa a simulação no intervalo [t_initial, t_end).

A cada passo, todos os fluxos são calculados primeiro (com os valores atuais dos Systems) e só então os Systems são atualizados — garantindo que a ordem dos fluxos não influencie o resultado (Euler explícito).

Esquema de execução assíncrona do run()
Parameters
t_initialInstante inicial (inclusive).
t_endInstante final (exclusive).

Implementa atualização assíncrona (Euler explícito):

  1. Para cada passo de tempo, todos os fluxos são avaliados primeiro e seus resultados guardados em um vetor temporário.
  2. Em seguida, os valores de source e target de cada fluxo são atualizados simultaneamente usando os resultados calculados no passo anterior, evitando que a ordem de avaliação influencie o resultado.
Parameters
t_initialInstante inicial da simulação (inclusive).
t_endInstante final da simulação (exclusive).
