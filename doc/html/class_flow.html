Detailed Description

Classe abstrata que representa um fluxo entre dois Systems.

Conecta um sistema de origem (source) a um sistema de destino (target) e define a taxa de transferência aplicada a cada passo de simulação. Equivale ao conceito de "Flow" nos diagramas de Vensim/Forrester.

Subclasses concretas devem implementar execute() com a equação do fluxo.
